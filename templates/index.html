Reliability analysis workspace

Inter-Rater Reliability Tool

Upload CSV or XLSX data, scan worksheet headings, select reliability pairs, confirm observations and raters, then calculate ICC, typical error, descriptive summaries, and agreement plots.

Optional user guide Read this first if you want a quick overview before uploading a workbook.

What the app does

  • reads CSV or XLSX workbooks
  • scans worksheet headings and guesses likely Test 1 / Test 2 pairs
  • lets you choose one or more reliability pairs to analyse
  • calculates ICC, 95% confidence intervals, descriptive summaries, and typical error
  • creates scatter and Bland-Altman plots plus Markdown and CSV exports

Recommended workbook structure

  • one row per observation or participant
  • one identifier column for the observation label
  • paired measurement columns named consistently, such as Metric Test 1 and Metric Test 2
  • numeric values in the measurement columns

How to use it

  1. upload the workbook
  2. choose the worksheet to analyse
  3. pick the observation ID column
  4. select one or more guessed reliability pairs
  5. review the ICC settings
  6. run the analysis and review each pair result section

What you get

  • pair-by-pair ICC estimates with 95% confidence intervals
  • summary statistics for each selected pair
  • typical error, bias, and limits of agreement
  • SVG/PDF figures
  • Markdown report and CSV export of analysed source data
